Accounting Application System

Advanced financial management solution with dual-database architecture

Streamlining financial oversight with MT940 processing, real-time analytics, and role-based access control

Comprehensive Financial Management System

Our accounting application revolutionizes financial tracking by providing intuitive tools for transaction management, real-time balance monitoring, and advanced MT940 file processing within a secure, role-based environment.

MT940 Processing

Advanced parsing and validation of MT940 files with seamless integration into both SQL and NoSQL databases, maintaining original transaction integrity while enabling custom descriptions.

Dual-Database Architecture

Hybrid data management system leveraging Microsoft SQL Server for relational data operations and MongoDB for flexible document storage, ensuring optimal performance and scalability.

Advanced Transaction Search

Powerful search capabilities by IBAN, transaction reference, or custom keywords with real-time filtering across millions of records for instant financial insights.

Role-Based Access Control

Granular authorization matrix with RWUD (Read, Write, Update, Delete) permissions tailored for different user roles, ensuring data security and compliance.

Entity Relationship Data Model

Core Entities

  • Transaction: Central entity with 30+ attributes including balances, dates, references
  • User: Role-based accounts with granular permissions (Admin/User)
  • Logs: Automated audit trails tracking all system activities

Implementation Strategy

  • Normalized SQL schema for financial transactions
  • Document-based NoSQL storage for MT940 raw files
  • Automated trigger-based logging system

Technical Specifications

Component Technology Specifications
Application Framework .NET Core 6.0 C# Desktop Application with WPF UI
Relational Database Microsoft SQL Server Normalized schema with ACID compliance
NoSQL Database MongoDB Document storage for MT940 files
Backend Node.js, Express, PostgreSQL
API Architecture RESTful design with resource-oriented endpoints
Authentication JWT & OAuth 2.0 Role-based access control (RBAC)
Backup Solution Microsoft Azure Automated weekly backups to cloud storage
System Requirements Windows 10/11 4GB RAM, 2GHz CPU, 500MB storage

Backup Strategy

Automated weekly backups to Microsoft Azure Blob Storage with 30-day retention policy and encryption-at-rest.

Role Management

Administrators have full RWUD permissions while standard users have restricted read/update access based on authorization matrix.

Authorization Matrix

Module Function User Administrator
MT940 Processing File Upload P RWUD
Validation P RWUD
Accounting Link to Cost Center R, U RWUD
Create Cost Center P RWUD
Transaction Categorization R, U RWUD
Data Views All Reports R RWUD

Permission Legend:

R: Read | W: Write | U: Update | D: Delete | P: Prohibited
